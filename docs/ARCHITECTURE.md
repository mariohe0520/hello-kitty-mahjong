# Hello Kitty 麻将 — 技术架构

## 概述
Hello Kitty 主题麻将游戏，可爱风格，单机 AI 对战。

## 技术栈
- **前端**: HTML + CSS + JS（`js/` + `css/` + `assets/`）
- **测试**: `test-qa.mjs` + `test-screenshots/`
- **部署**: GitHub Pages

## 核心系统
| 系统 | 说明 |
|------|------|
| 牌局逻辑 | 发牌/摸牌/打牌/吃碰杠胡 |
| AI 对手 | 电脑出牌策略 |
| UI 渲染 | 手牌展示/牌桌布局 |
| 动画 | 出牌/胡牌动效 |

## 约束
- 目标用户：芒果过敏（不会打麻将也能玩）
- 教程/引导必须友好
- Hello Kitty 视觉主题
- 触控操作优先（手机）
- 纯前端，状态存内存（单局制）
